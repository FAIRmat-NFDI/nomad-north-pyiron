# pyiron - NORTH tool

This directory contains the configuration for the pyiron NORTH (NOMAD Remote Tools Hub) tool and NOMAD plugin.

## Quick start

The pyiron NORTH tool provides a containerized environment defined in `NORTHtool` definition and a `NORTHToolEntryPoint`.

## Documentation

For comprehensive documentation on creating and managing NORTH tools, including e.g.,

- Entry point configuration and `NORTHTool` API
- Docker image structure and best practices
- Dependency management
